PÁGINA PRINCIPAL

Balatro

Implementa un programa que detecte y asigne puntos a una mano de cartas estilo póker, dándole más puntos a combinaciones más raras.

$ ./balatro
Carta 1 (ej. AH para As de Corazones): 9H
Carta 2: TH
Carta 3: JH
Carta 4: QH
Carta 5: KH

Escalera de color encontrada: 5000 puntos.

Reglas

  • Cada carta es representada por dos caracteres:
    • El valor: 2–9, T (10), J, Q, K, A
    • El palo: H (♥️), D (♦️), S (♠️), C (♣️)
    Ejemplo: "AH" significa As de Corazones.

    • Combinaciones y puntos

    Mano de Póker Puntos
    Escalera Real 10000
    Escalera de color 5000
    Póker (cuatro iguales) 3000
    Full House 2000
    Color (Flush) 1500
    Escalera 1000
    Trío (tres iguales) 500
    Dos pares 300
    Un par 100
    Carta alta 50

    Consideraciones

    • El programa debe validar que:
      • Las cartas sean únicas.
      • Tengan formato correcto (valor válido + palo válido).
    • El programa debe ser modular con funciones como:
      • es_escalera()
      • es_color()
      • contar_valores()
    • Usa structs para representar cartas: 
      typedef struct {
        int valor;  // 2–14 (donde T=10, J=11, Q=12, K=13, A=14)
        char palo;  // 'H', 'D', 'S', 'C'
        } carta;

    Empezando

    Abre VS Code

    Comienza haciendo click dentro de la ventana de tu terminal, luego ejecuta cd por sí mismo. 

    $

    Haz clic dentro de esa ventana de terminal y luego escribe

    wget https://cdn.cs50.net/2025/x/balatro.zip

    seguido de Enter para descargar un ZIP llamado balatro.zip en su espacio de código.

    Ahora ejecuta

    unzip balatro.zip

    para crear una carpeta llamada balatro. Ya no necesita el archivo ZIP, por lo que puede ejecutar

    rm balatro.zip

    y responda con "y" seguido de Enter en el aviso para eliminar el archivo ZIP que descargó.

    Ahora escribe

    cd balatro

    para moverte hacia (es decir, abrir) ese directorio. Tu terminal debería parecerse a lo siguiente.

    balatro/ $

    Si todo salió bien, debes ejecutar.

    ls

    y ver un archivo llamado balatro.c. La ejecución code balatro.c debería abrir el archivo donde escribirá su código para este conjunto de problemas.

    Implementación

    Tu archivo balatro.c debe implementar las siguientes funciones:

    bool es_par(carta mano[]);
        bool es_dos_pares(carta mano[]);
        bool es_trio(carta mano[]);
        bool es_escalera(carta mano[]);
        bool es_color(carta mano[]);
        bool es_full_house(carta mano[]);
        bool es_poker(carta mano[]);
        bool es_escalera_color(carta mano[]);
        bool es_escalera_real(carta mano[]);
        
        int calcular_puntos(carta mano[]);

    Evaluación

    Tu programa debe:

    • Validar entradas inválidas.
    • Detectar correctamente la mano.
    • Asignar el puntaje correcto.
    • Imprimir un mensaje como: Escalera de color encontrada: 5000 puntos.

    ¿Cómo entregarlo?

    Abre una terminal y ejecuta lo siguiente:

    submit50 cs50/problems/2025/x/balatro
